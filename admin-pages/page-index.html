ページと基本フォーム・一覧・詳細

このページはヘッダーの「ページ」から開く一覧です。「基本フォーム」「一覧」「詳細」の3画面がこのグループに紐づきます。サイドバーでは「ページ」を親に、同じ3リンクが子として並びます。

ページに紐づく画面

edit_document 基本フォーム（サンプル）
`form-basic.html` 入力レイアウトの見本
view_list 一覧（サンプル）
`list-basic.html` テーブル・カード型リスト
description 詳細（サンプル）
`detail-basic.html` 一覧から遷移する詳細画面
